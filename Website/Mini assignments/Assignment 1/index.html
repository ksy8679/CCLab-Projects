Expect the Unexpected

  • Bongo Cat
  • Kimberly Yau
  • 2022

Expect the Unexpected is a creative piece that is coded within p5.js as well. This project explores the basics of drawing with code (JavaScript). This mini assignment consists of replicating an image or objects that we have in our daily lives with code. The picture I decided to recreate was my door, but it seemed too simple at the beginning. It may still look very simple with the door closed, but when a user clicks anywhere on the screen, the door opens. There, it reveals the Bongo Cat behind the door, which is why it is named: Expect the Unexpected. It's a small surprise for the person interacting with this piece.
