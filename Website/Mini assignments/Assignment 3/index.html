Sketchpad

  • Interactive Drawing
  • Kimberly Yau
  • 2022

This Sketchpad is an interactive drawing coded in the p5.js platform. This piece consists of implementing code that engaged the user through different interactions. Within this sketchpad, there is an initial drawing (a heart), and the drawing's color changes depending on which side of the screen the user's mouse is located in. However, the user can turn this image into a sketchpad by pressing the key 'd'. Within the sketchpad, the user can draw with their mouse as long as the mouse if pressed, and create shapes such as circles, rectangles and squares.
